# Geometry Shapes Task

Solution for a geometry shapes task implemented in Python.

## Description

The program reads shape descriptions, calculates their areas, and prints the results.

Supported shapes:
- square
- rectangle
- circle
- triangle

## Project structure

geometry-shapes-task
├── task.py
├── task1.py
├── test_shapes.py
└── README.md

## Example

width = abs(x1 - x2)
height = abs(y1 - y2)

We use abs() to ensure the width and height are always positive regardless of the order of coordinates.
